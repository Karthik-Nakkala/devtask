# 📱 React JS - Intern/Fresher Qualifier Task

This project is a **pixel-perfect implementation** of the given Adobe XD design using **React JS**, styled with **TailwindCSS**, and enhanced with **Framer Motion** animations.  
It was built as part of the **internship/fresher qualifier task**.

- **React JS** – Component-based UI

- **React Router** – Seamless navigation
- **Vercel** – Hosting & deployment
